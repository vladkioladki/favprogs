## 🎮 Ігри
* **Gale** - програма для модифікації ігор в основному на unity (Lethal Company, Risk Of Rain 2, Among Us etc.)
* **Limo** - FOSS мод менеджер для ігор від Bethesda
* **Rare** - Epic Games Launcher але без інтерфейсу що знищить вашу оперативну пам'ять. Для тих кому лінь ставити офіційний клієнт.
* **Vortex** - нексус лаунчер гавно.
* **Hydra Launcher** ⭐ - НАЙКРАЩИЙ спосіб безкоштовно скачувати ігри. Є соціальні функції, додавання джерел для скачування та купа всього іншого. Маст хев для тих хто ще досі качає ігри через браузер.
* **Itch.io** - ваші любімі інді ігрушкі
* **Heroic Game Launcher** - Один з найкращих лаунчерів для Лінукс чи МакОС.
* **PCXSense** - FOSS программа для тих в кого dualsense

## 🕹️ Емулятори
* **Ludo** 🤷‍♂️ - в теорії якийсь рескін ретроарчу
* **Mandarine** 🤷‍♂️ - форк емулятору Citra (3DS)
* **PCSX2** ⭐ - емулятор PS2
* **RPCS3** 🤷‍♂️ - емулятор PS3
* **Mesen** - Емулятор NES,SNES,Game Boy без потреби встановлювати (просто .exe файл)

## 🟩 Minecraft
* **Prism Launcher** - найкращий лаунчер для майнкрафту
* **Freesm Launcher** - найкращий лаунчер для майнкрафту для тих в кого немає грошей
* **Shatteredprism launcher** ⭐ - найкращий лаунчер для майнкрафту для тих в кого немає грошей і для тих в кого є гроші. Або просто дуже покращена версія оригінального лаунчеру Prism

## 🌐 Браузери
* **Helium** ⭐ - Браузер на базі Chromium що має AdBlock з встановлення. Найкращий браузер для тих хто любить Хром.
* **Firefox** - Легендарний браузер, користувався ним довше всього. Краще ставити ![Firefox-ESR](https://www.firefox.com/en-US/browsers/enterprise/)
* **Zen Browser** - Браузер на основі Firefox, зараз в розробці тому можуть бути баги. Для естетів і тих хто користується вертикальними вкладками.
* **Floorp** ⭐ - Firefox на максималках, кастомізація, підтримка розширень Chrome і ще багато всього.
* **Vivaldi** - Для максимальних естетів. Дуже непоганий браузер (Мій топ 2 серед браузерів на Android) але пропрієтарний та доволі тяжкий для системи.
* **Cromite** 📱 - Найкращий браузер для Android. Є блокування реклами та багато налаштувань, мобільний хром здорової людини.
